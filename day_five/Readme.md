# 第五天的题目

## 题目 1：简单的数字分类

编写一个程序，接收用户输入的一个整数，然后使用 switch 语句判断该整数的范围，并输出相应的分类：

-   小于 0：负数

-   等于 0：零

-   1 到 10：小于等于 10

-   大于 10：大于 10

**示例：**

```bash
输入：5
输出：5 属于 1 到 10 的范围
```

# 题目 2：九九乘法表

编写一个程序，使用 for 循环打印 1 到 9 的乘法表。

示例输出：

```bash
1 x 1 = 1
1 x 2 = 2
...
9 x 9 = 81
```

# 题目 3：计算阶乘

编写一个**函数** `factorial(n int) int`，计算并返回一个整数 n 的阶乘（n!）。在主函数中，接收用户输入的一个非负整数，并输出它的阶乘。

**示例：**

```bash
输入：5
输出：5 的阶乘是 120
```
